## 关键 UI 设计要点

### 1. 区间范围展示（Min/Typ/Max）

```tsx
// 使用颜色编码区分三个值
<div className="flex items-center gap-2">
  <span className="text-red-600 font-mono">Min: 8.52V</span>
  <span className="text-gray-400">|</span>
  <span className="text-blue-600 font-mono font-semibold">Typ: 9.00V</span>
  <span className="text-gray-400">|</span>
  <span className="text-orange-600 font-mono">Max: 9.48V</span>
</div>
```

**设计原则**：
- **红色 (Min)**：最坏情况下限
- **蓝色 (Typ)**：典型值，加粗强调
- **橙色 (Max)**：最坏情况上限
- 使用等宽字体 (font-mono) 确保对齐

### 2. 警告提示层级

```tsx
// 致命警告（红色，SOA 超限）
<div className="bg-red-50 border-l-4 border-red-500 p-4">
  <span className="text-2xl">🔥</span>
  <div className="font-semibold">致命警告</div>
  <div>MOSFET 热应力超限！实际 I²t = 2500 A²·s，极限 = 2000 A²·s</div>
</div>

// 一般警告（黄色，浪涌电流超限）
<div className="bg-yellow-50 border-l-4 border-yellow-500 p-4">
  <span className="text-2xl">⚠️</span>
  <div className="font-semibold">注意</div>
  <div>浪涌电流 35A 超过连接器限制 30A</div>
</div>
```

**层级设计**：
- **Level 1 (致命)**：红色 + 🔥，阻塞性问题（MOSFET 会烧毁）
- **Level 2 (警告)**：黄色 + ⚠️，需要注意但不致命
- **Level 3 (提示)**：蓝色 + 💡，优化建议

- 鼠标悬停显示备选阻值（Tooltip）

### 4. 迟滞电流偏移可视化

```tsx
<div className="bg-blue-50 border border-blue-200 p-4">
  <div className="font-semibold">⚡ 迟滞电流导致的关断电压偏移</div>
  <div className="text-sm mb-3">

    通过 R10 产生电压降，导致实际关断电压低于理想分压值。
  </div>
  
  {/* 偏移量展示 */}
  <div className="flex gap-4 font-mono text-xs">
    <div>Min 偏移: <span className="text-red-600">-0.384V</span></div>
    <div>Typ 偏移: <span className="text-blue-600">-0.485V</span></div>
    <div>Max 偏移: <span className="text-orange-600">-0.602V</span></div>
  </div>
  
  {/* 公式说明 */}
  <div className="mt-3 bg-blue-100 p-2 rounded text-xs">
    💡 公式：ΔV = I_HYS × R10
    <br/>来源：Datasheet SNVS628H Section 8.2.3.2.1, Page 31
  </div>

```

**设计原则**：
- 蓝色背景突出物理现象
- 负号明确表示电压下降
- 公式 + datasheet 来源增强可信度

## 交互反馈

1. **加载状态**：旋转的圆环动画
2. **悬停效果**：表格行高亮 (hover:bg-gray-50)
3. **Tooltip**：鼠标悬停显示详细信息
4. **折叠面板**：高级参数可展开/收起
5. **按钮状态**：禁用时变灰 (disabled:bg-gray-400)

## 可访问性 (A11y)

- 所有颜色编码同时使用图标（✅❌⚠️）
- 表单字段带 label 和 aria-label
- 键盘导航支持 (Tab 键)
- 对比度符合 WCAG AA 标准

## 总结

核心设计理念：
1. **信息层级清晰**：致命警告 > 一般警告 > 正常结果
2. **数据可视化**：区间范围用颜色编码，一目了然
3. **交互友好**：Tooltip、悬停效果、折叠面板
4. **专业可信**：公式来源、datasheet 引用、精确计算
